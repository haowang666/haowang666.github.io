近日因为研究需要开始大规模转向Python。作为一个资深R使用者，不得不说有些习惯要注意一下。

Python中的赋值是动态的。

```Python
a = 6
b = a
a = 7
print(b)
###########
---> return 7
```
在R里面只看顺序

```R
a <- 6
b <- a
a <- 7
print(b)\
###########
---> return 6

```
